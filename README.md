# AWS NestJS Microservice
## Description

A NestJS Microservices deployed using Docker, k8s, AWS Load Balancer, eksctl, etc.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev _auth || orders || billing_

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
